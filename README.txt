Инструкция по запуску
Данный проект представляет собой утилиту для обработки файлов, которая сортирует различные типы данных (целые числа, вещественные числа, строки) и записывает их в соответствующие файлы.


Шаги:
- Склонируйте репозиторий на ваш компьютер

- Соберите проект с помощью Maven:
mvn clean package

- После сборки проекта вы можете найти JAR-файл в директории target. Запустите утилиту, указав путь к JAR-файлу и необходимые аргументы командной строки. Например:
java -jar target/testTask-1.0-SNAPSHOT.jar -s -a -p sample_ in1.txt in2.txt

Желательно использовать абсолютные пути к jar файлу и к txt файлам


Опции командной строки:
-o задать путь для результатов
-p задать префикс имен выходных данных
-a задать режим добавления в существующие файлы
-s краткая статистика
-f полная статистика
Примечание: Если опция -o не указана, файлы результатов будут сохранены в текущей директории.

Зависимости:
Apache Commons CLI 1.4

Информация о сборке:
Проект собирается с использованием Apache Maven. Плагин Maven Shade используется для создания фатального JAR-файла с зависимостями.
